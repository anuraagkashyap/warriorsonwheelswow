A - Description - top

This script Paypal Payment Pack allows you to have a quick & easy payment terminal for your clients. Only need to put this script into any directory on your hosting account, edit configuration file and give the link to your customers.

The pack has a set of solutions that allow you to:
Buy a single element
Buy multiple items
Make a Subscription
Make a donation
"Buy me a beer"

Demo WARRIORS ON WHEELS

B - Installation - top

1- Select the terminal you want to use. Open "paypal_config.php" and edit the values ​​that you want. If you want to use test mode you will have to change the value of $use_paypal_sandbox = true; In each line you will see a description of each parameter.

2- Upload all files (including paymentpack folder) on your server ... and you will be the payment system ready for use.

3-If you wan use MySql:
a-Config file paypal_config.php for MySql settings
b-Use install_mysql.php to install table
c-Delete install_mysql.php from server



C - Customize colors - top

The Payment Pack has 4 predefined color styles. To activate any of the 4 you just have to uncomment the one you want to use. 

        
    	< link rel="stylesheet" href="paymentpack/css/style_green.css" media="screen" />
    	< link rel="stylesheet" href="paymentpack/css/style_orange.css" media="screen" />
    	< link rel="stylesheet" href="paymentpack/css/style_red.css" media="screen" />
    	< link rel="stylesheet" href="paymentpack/css/style_blacknwhite.css" media="screen" />
It's easy you can also change the colors and the message "Thanks", just edit the file paymentcomplete.php


D - PayPal SandBox - top

If you want to test the installed system can activate SandBox.
To activate the Paypal SandBox within config.php you have to replace this $payaltest = false;, for this $payaltest = true;

Create an account in Paypal testing
First you have to register in https://developer.paypal.com/

Register an account Paypal SandBox testing to Sell / Seller
1 - Go to Test Accounts
2 - Click Preconfigured
3 - Click on Account Type "Seller (Use to represent ...)"
4 - Fill form
5 - Create account button

Register an account Paypal SandBox testing to Buy
1 - Go to Test Accounts
2 - Click Preconfigured
3 - Click on Account Type "Buyer (Use to represent your customer's...)"
4 - Fill form
5 - Put 'virtual money' in Account balance (important!)
5 - Create account button
